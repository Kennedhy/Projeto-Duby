Perguntas Frequentes

Como funciona a conciliação financeira?

A conciliação financeira é o processo de comparar transações financeiras registradas em diferentes sistemas para garantir que os valores estejam corretos. Nosso sistema automatiza esse processo, identificando discrepâncias e ajudando você a resolvê-las.

Como importar dados bancários?

Para importar dados bancários, acesse a seção 'Contas' no menu lateral, selecione a conta desejada e clique no botão 'Importar Dados'. Você pode fazer upload de arquivos CSV, OFX ou extratos bancários em outros formatos.

O que significam as discrepâncias no dashboard?

Discrepâncias são diferenças encontradas entre os valores esperados e os valores reais nas transações. Elas podem indicar erros, fraudes ou problemas de processamento. No dashboard, você pode visualizar todas as discrepâncias identificadas e tomar ações para resolvê-las.

Como resolver uma discrepância?

Para resolver uma discrepância, clique nela no dashboard para ver os detalhes. Você pode então escolher uma das ações recomendadas ou registrar uma justificativa para a diferença. Após resolver, a discrepância será marcada como resolvida e movida para o histórico.

Como adicionar um novo usuário ao sistema?

Para adicionar um novo usuário, acesse a seção 'Usuários' no menu lateral e clique no botão 'Adicionar Usuário'. Preencha as informações necessárias, defina as permissões apropriadas e clique em 'Salvar'.

Como configurar uma nova adquirente?

Para configurar uma nova adquirente, vá até a seção 'Contas', selecione a aba 'Adquirentes' e clique em 'Adicionar Adquirente'. Preencha os dados solicitados, incluindo as taxas e condições de pagamento, e clique em 'Salvar'.
